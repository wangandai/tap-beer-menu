# tap-beer-menu
For lazy people to know whether its the day to head down to any bar for a beer fix

# Requirements
create a `key.json` file with the contents as such
```
{
  "bot_token": "<TELEGRAM_BOT_TOKEN>"
}
```

# Start service
cd to root folder, then run:
```
bash start.sh
```

# Stop service
cd to root folder, then run:
```
bash stop.sh
```

# Clear logs
```
bash clearlogs.sh
```
